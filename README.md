# Exodar

Meteor Client addon for Minecraft 26.1.2.

## Features

**Modules** (Exodar category):
- ESprint
- ETool
- AimAssist
- TriggerBot
- AutoClicker
- LegitAura
- Penetration

## Requirements

- JDK 25
- Minecraft 26.1.2 + Fabric Loader 0.19.2
- Meteor Client 26.1.2

## Build

```
./gradlew build
```

Output: `build/libs/exodar-addon-0.1.0.jar`.

## Install

Drop the built JAR into your Fabric mods folder alongside Meteor Client.
