# Exodar (1.21.11 branch)

Meteor Client addon source for Minecraft 1.21.11.

## Download

| Minecraft Version | Exodar version | Dependencies |
|-------------------|----------------|--------------|
| 1.21.11 | [1.0.1](https://github.com/ShaunaAyala/Exodar-Addon/releases/download/v1.0.1/Exodar-Addon-1.0.1-1.21.11.jar) | Fabric Loader 0.16.10+, Meteor Client 1.21.11 |

For the 26.1.2 build, see the [main branch](https://github.com/ShaunaAyala/Exodar-Addon).

## Features

**Modules** (Exodar category):
- ESprint
- ETool
- AimAssist
- TriggerBot
- AutoClicker
- LegitAura
- Penetration

## Build

```
./gradlew build
```

Output: `build/libs/exodar-addon-1.0.1.jar`

Toolchain:
- JDK 21
- Loom 1.14.10
- Mojang official mappings
